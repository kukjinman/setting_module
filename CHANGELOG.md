# Changelog

이 저장소의 릴리스 변경 사항을 기록합니다. 버전은 [Semantic Versioning](https://semver.org/)을 따르며, Git 태그는 `v<version>` 형식을 사용합니다.

## [1.0.0] - 2026-07-22

- 저장, 인증, 오디오 및 네이티브 햅틱을 제공하는 Shared Core Module 추가
- Intro, Login, Home, Gameplay, Collection 및 Settings 공용 흐름 추가
- 프로필과 하단 메뉴가 포함된 게임별 커스터마이징 가능 홈 화면 추가
- English/한국어 Locale, 공용 String Table 및 오른쪽 하단 언어 선택 UI 추가
- Apple Game Center와 Google Play Games 로그인 공급자 경계 추가
- 태그 기반 UPM 설치와 저장소 단위 버전 관리 체계 추가
